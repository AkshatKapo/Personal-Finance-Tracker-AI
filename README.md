# Personal Finance Tracker AI

A full-stack personal finance tracking system that helps users monitor spending, categorize transactions, and visualize financial trends.

## Features

- Track income and expenses
- Categorize transactions
- View spending trends through dashboards
- Budget tracking and financial summaries

## Tech Stack

Frontend  
- React

Backend  
- Node.js  
- Express

Database  
- PostgreSQL

Infrastructure  
- Docker

## AI Insights

The system was designed to generate spending insights using an AI API.  
Since API keys are not included in the repository and quotas may vary, the current version uses **mocked responses** to demonstrate the intended AI insight functionality.

The architecture allows the AI module to be easily reconnected by adding an API key to the backend service.

## Project Structure

frontend/ – React UI  
backend/ – Node.js API and data logic
